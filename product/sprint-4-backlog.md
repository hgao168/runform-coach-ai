## 三、优先级排序（按用户价值 × 实现成本）

| 优先级 | 差距项 | 理由 |
|--------|--------|------|
| **P0** | Web History + Profile + 视频分析结果持久化 | Web 目前无任何留存能力，上传分析后数据丢失，是最致命短板 |
| **P0** | Android RunSession 回放 | iOS 已交付的高价值功能，Android 用户无法查看跑步回放，留存差 |
| **P1** | 微信小程序周洞察报告 | 小程序核心活跃用户缺少训练洞察，影响留存和分享裂变 |
| **P1** | Web 训练计划生成 | Web 用户无训练计划入口，功能断层严重 |
| **P1** | Web Compare 对比 | 差异化功能未覆盖 Web，错失获客转化 |
| **P2** | 微信小程序分享卡片完善 | 已有基础 Canvas，需优化为独立分享卡片（含小程序码） |
| **P2** | Android 独立分享卡片 | 当前仅洞察内可分享，需支持历史/分析结果独立生成分享图 |
| **P2** | 全平台营销维护 | 上线后 ASO 迭代、社交媒体内容、用户反馈闭环 |
