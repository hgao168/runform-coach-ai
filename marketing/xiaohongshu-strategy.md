## 一、小红书平台分析

### 1.1 为什么小红书是 RunForm 的核心渠道？

```
┌──────────────────────────────────────────────────────────────┐
│              小红书 = 跑步人群的「种草引擎」                    │
│                                                              │
│  用户画像（与 RunForm 核心 TA 高度重合）：                      │
│  · 女性 70% / 男性 30%（跑步内容男女比例逐渐均衡）              │
│  · 25-35 岁为主，一二线城市                                    │
│  · 月活 3亿+，日均搜索 3亿次                                   │
│                                                              │
│  跑步话题数据：                                                │
│  · #跑步打卡 浏览量 30亿+                                      │
│  · #马拉松训练 浏览量 8亿+                                     │
│  · #跑步姿势 浏览量 3亿+                                       │
│  · #跑步受伤 浏览量 2亿+                                       │
│  · #跑步教练 浏览量 5,000万+                                   │
│                                                              │
│  关键洞察：跑步「姿势」「伤病」「教练」三个维度的搜索量巨大     │
│  但高质量内容供给严重不足——这是 RunForm 的内容蓝海。            │
└──────────────────────────────────────────────────────────────┘
```

### 1.2 小红书搜索行为特征

用户在小红书的搜索行为不同于传统搜索引擎：
- **「问题型」搜索**：「跑步膝盖疼怎么办」「正确跑步姿势图解」
- **「对比型」搜索**：「跑步前后对比」「正确跑姿 vs 错误跑姿」
- **「工具型」搜索**：「跑步姿势分析 App」「跑步教练推荐」
- **「打卡型」浏览**：不是搜索，而是日常刷跑步相关笔记

## 三、笔记 SEO 策略

### 3.1 标题关键词策略

**标题公式**：`[痛点/好奇心钩子] + [数据/对比] + [解决方案暗示]`

**标题模板库**：

| 类型 | 模板 | 示例 |
|------|------|------|
| 痛点 + 数字 | 「[数字]% 的跑者都不知道自己的[问题]」 | 「90%的跑者都不知道自己的步频太低——我用 AI 测了才知道」 |
| 对比 + 惊讶 | 「我的[指标] vs [精英]的差距，看完沉默了」 | 「我的步频 vs 基普乔格的步频，看完差距沉默了」 |
| 教程 + 工具 | 「[问题]？用这个方法 30 秒自测」 | 「膝盖痛是不是跑姿问题？用这个方法 30 秒自测」 |
| 打卡 + 进步 | 「[时间]打卡第[N]天，[指标]从[A]到[B]」 | 「步频打卡第 30 天，从 164 到 178 的变化太大了」 |
| 清单 + 收藏 | 「[数字]个跑姿自检动作，跑前必做」 | 「5 个跑姿自检动作，跑前花 2 分钟从此不受伤」 |
| 悬念 + 揭秘 | 「跑了[N]年才知道，原来我一直[错误]」 | 「跑了 3 年才知道，原来我一直在用错误的姿势伤膝盖」 |

### 3.2 关键词矩阵

| 类别 | 高搜索量词 | 中搜索量词(蓝海) | 长尾精准词 |

## 五、视觉设计规范

### 5.1 封面图设计原则

| 要素 | 规范 |
|------|------|
| 尺寸 | 3:4 竖版 (1080 × 1440px) |
| 字体 | 粗体无衬线，标题字号 ≥ 48pt |
| 配色 | 深色背景 + Mint Green (#00C9A7) 高亮 + 白色文字 |
| 元素 | 数据圈注、前后对比箭头、人物剪影 |
| 品牌 | 右下角小 Logo，不抢视觉 |

### 5.2 配图模板库

| 类型 | 模板结构 | 使用场景 |
|------|---------|---------|
| 对比图 | 左(改善前) vs 右(改善后)，中间箭头 + 数据变化 | 跑姿对比、步频改善 |
| 数据卡 | 大数字 + 单位 + 简短说明 + 绿/黄/红色评级 | 步频、触地时间、垂直振幅 |
| 自测清单 | 6 宫格，每格一个动作图示 + 一句话说明 | 跑姿自测、伤病预防 |
| 案例故事 | 人物剪影 + 痛点文字 + RunForm 界面截图 + 改善结果 | 用户故事 |
| 教程步骤 | 1-2-3 步骤编号 + 配图 + 简短说明 | 产品使用教程 |

### 5.3 不可用素材

- ❌ 纯文字截图（小红书算法降权）
- ❌ App Store 截图直接搬运（太像广告）
- ❌ 低分辨率图片（至少 1080px 宽）
- ❌ 真人面部未打码（除非已获授权）

## 六、KOC 合作方案

### 6.1 KOC 筛选标准

```
┌──────────────────────────────────────────────────────────────┐
│              RunForm KOC 合作筛选漏斗                          │
│                                                              │
│  初筛池（50 人）                                              │
│  ├─ 小红书跑步相关笔记 ≥ 10 篇                                │
│  ├─ 粉丝 1,000 - 50,000                                      │
│  └─ 近 30 天有更新                                            │
│        ↓                                                     │
│  精选池（15 人）                                              │
│  ├─ 互动率（赞藏评/粉丝）≥ 5%                                 │
│  ├─ 内容类型：跑步打卡 / 马拉松训练 / 运动康复 / 跑步干货       │
│  └─ 调性匹配：认真跑者，非纯颜值/穿搭/泛健身                    │
│        ↓                                                     │
│  合作池（目标 5 人）                                          │
│  ├─ 接受合作邀请                                              │
│  ├─ 使用 RunForm 后愿意产出真实体验笔记                        │
│  └─ 提供专属小程序码追踪转化                                    │
└──────────────────────────────────────────────────────────────┘
```

### 6.2 KOC 分级合作模式

| 级别 | 粉丝量 | 合作方式 | 激励 | 预期笔记数 |
|------|--------|---------|------|-----------|
| **A 级** | 10,000 - 50,000 | 深度体验 + 跑姿改善记录 | ¥500 稿费 + 1 年 Pro 会员 | 2-3 篇/人 |
| **B 级** | 3,000 - 10,000 | 产品体验 + 真实测评 | ¥200 稿费 + 6 个月 Pro 会员 | 1-2 篇/人 |
| **C 级** | 1,000 - 3,000 | 免费体验 + 自愿发布 | 3 个月 Pro 会员 | 1 篇/人（非强制） |

### 6.3 KOC 合作 Brief

**提供给 KOC 的内容**：
1. RunForm Pro 免费体验账号
2. 产品核心卖点一页纸（非脚本，让 KOC 用自己的话表达）
3. 跑姿分析示例报告（供参考，不强制使用）
4. 专属小程序码（含追踪参数，如 `?source=xhs_koc_001`）
5. 建议内容方向（3 选 1）：
   - 「我的跑姿分析报告 + 最惊讶的发现」
   - 「用 RunForm 实时教练跑步的真实体验」
   - 「我和基普乔格跑姿对比，差距有多大」

**不强制的内容**：
- 不要求正面评价（真实 > 好评）
- 不要求特定文案
- 不要求发布时间（给 2 周窗口期）
