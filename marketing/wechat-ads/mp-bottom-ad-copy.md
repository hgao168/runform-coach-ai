## 广告位规格

| 参数 | 规格 |
|------|------|
| 广告位类型 | 公众号底部广告（MP 广告） |
| 展示位置 | 已关注公众号推文底部、留言区上方 |
| 图片尺寸 | 960×540 px（16:9 大图卡片） |
| 文案限制 | 标题 ≤ 14 字，描述 ≤ 30 字 |
| 落地页 | 微信小程序（RunForm 分析页） |
