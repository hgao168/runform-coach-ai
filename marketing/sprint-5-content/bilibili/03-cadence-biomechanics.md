#### Part 2：180是怎么来的？（1:00-2:30）
- 1984年洛杉矶奥运会，跑步教练Jack Daniels观察了50名精英跑者
- 发现他们的步频在180-200之间，平均约180
- 关键点：这些是比赛配速下的精英跑者（身高多在170-178cm）
- 问题：精英选手比赛配速3:00/km的步频，不等于你配速6:00/km的步频
- 画面：尺子对比 —— "就像不能用姚明的投篮姿势教小学生打球"
- 引用文献（至少1-2篇）：步频研究的关键论文

#### Part 3：决定最佳步频的4个因素（2:30-5:00）
- 动画 + 图表辅助

```
【因素1：身高】
- 高个子 → 步幅自然大 → 步频自然低
- 矮个子 → 步幅自然小 → 步频自然高
- 动画：175cm vs 185cm跑者同配速下的步频对比
- 粗略公式：最佳步频 ≈ 190 - (身高cm - 170) × 0.5

【因素2：配速】
- 配速越快 → 步频越高
- 轻松跑 6:30/km → 步频160-170可能也OK
- 比赛跑 4:00/km → 步频180+才合理
- 图表：不同配速对应的推荐步频范围（引用研究数据）

【因素3：跑步经验】
- 新手 → 步频建议偏低起步，先保证动作稳定性
- 进阶跑者 → 可逐步提升目标步频
- 精英跑者 → 步频自然趋向高效区间
- "强行把新手步频拉到180，可能导致动作变形"

【因素4：肌肉力量与柔韧性】
- 核心力量 + 髋关节灵活性 → 支持更高步频
- 力量不足时强行高频 → 代偿动作 → 受伤
```
