# Running Form Analysis Framework — Metrics & Coaching Suggestions

This document summarizes the professional coaching feedback into a structured system you can use for:

* your running analysis app
* AI posture detection
* coaching reports
* automated recommendations
* strength/stretch prescription engine

---

# 1. Core Running Metrics to Capture

## A. Head & Neck

| Metric                | What to Detect            | Possible Issue      |
| --------------------- | ------------------------- | ------------------- |
| Head tilt angle       | Head alignment vs torso   | Head tilted in neck |
| Forward head position | Ear ahead of shoulder     | Head protruding     |
| Gaze direction        | Looking downward too much | Head bent forward   |

### Suggested Corrections

* Chin tuck
* “Look ahead / enjoy the view”
* Stack head above torso

---

# B. Shoulder & Upper Body

| Metric               | What to Detect             | Possible Issue       |
| -------------------- | -------------------------- | -------------------- |
| Shoulder elevation   | Shoulder height during run | Raised shoulders     |
| Shoulder symmetry    | Left/right imbalance       | Compensation pattern |
| Upper back curvature | Thoracic rounding          | Rounded upper back   |
| Lumbar extension     | Excessive arch             | Hollow lower back    |
| Torso lean angle     | Lean from ankles vs hips   | Hip hinge posture    |

### Suggested Corrections

* Relax shoulders
* Rib cage down
* Lean from ankles
* Core engagement
* Chest lifted

---

# C. Hip Mechanics

| Metric                | What to Detect        | Possible Issue    |
| --------------------- | --------------------- | ----------------- |
| Pelvic drop           | Opposite hip drops    | Weak glute medius |
| Hip position          | Hips behind body      | Sitting posture   |
| Hip sway              | Side-to-side movement | Instability       |
| Knee-to-hip alignment | Knee collapse inward  | Dynamic valgus    |

### Suggested Corrections

* Single-leg balance work
* Glute strengthening
* Cadence increase
* Core tension drills
* Wider foot placement

---

# D. Cadence & Ground Contact

| Metric               | What to Detect     | Possible Issue  |
| -------------------- | ------------------ | --------------- |
| Cadence (SPM)        | Steps per minute   | Too low cadence |
| Ground contact time  | Time on ground     | Overstriding    |
| Vertical oscillation | Excessive bouncing | Energy waste    |
| Running noise        | Heavy landing      | Poor stiffness  |

### Suggested Corrections

* Increase cadence
* Smaller steps
* Faster arm rhythm
* Quick foot pickup
* Quiet running drills

---

# E. Arm Mechanics

| Metric                    | What to Detect      | Possible Issue  |
| ------------------------- | ------------------- | --------------- |
| Arm swing direction       | Sideways crossing   | “Knitting”      |
| Backward drive            | Lack of rear drive  | Weak propulsion |
| Elbow angle               | Too open or fixed   | Tension         |
| Shoulder-arm independence | Whole torso rotates | Compensation    |

### Suggested Corrections

* Drive elbows backward
* Relax wrists
* Keep shoulders loose
* Compact arm angle
* “Movement happens behind”

---

# F. Knee Mechanics

| Metric                     | What to Detect     | Possible Issue  |
| -------------------------- | ------------------ | --------------- |
| Knee drive height          | Low knee lift      | Weak stride     |
| Knee drive speed           | Slow recovery      | Poor elasticity |
| Knee flexion during stance | Excessive collapse | Weak stiffness  |

### Suggested Corrections

* High knee drills
* Fast cadence drills
* Triple extension drills
* Active arm use

---

# G. Foot Strike & Placement

| Metric               | What to Detect        | Possible Issue |
| -------------------- | --------------------- | -------------- |
| Heel strike severity | Landing ahead of body | Braking forces |
| Forefoot loading     | Excessive toe landing | Calf overload  |
| Foot placement       | Foot ahead of hips    | Overstride     |
| Foot symmetry        | Left/right imbalance  | Compensation   |

### Suggested Corrections

* Foot under hips
* Shorter stride
* Faster cadence
* Circular leg motion
* Midfoot transition

---

# H. Push-Off / Propulsion

| Metric                  | What to Detect    | Possible Issue  |
| ----------------------- | ----------------- | --------------- |
| Hip extension           | Lack of extension | Weak propulsion |
| Ankle extension         | Weak toe-off      | Low power       |
| Triple extension timing | Poor sequencing   | Inefficiency    |

### Suggested Corrections

* Plyometrics
* Bounding
* Skip drills
* Sprint mechanics drills
